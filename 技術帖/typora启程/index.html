常用快捷键

  1. 申明：第一部分内容涵盖了word上一些基础操作的使用，如word菜单栏上的加粗(B)、斜体(I)、下划线(U)^10，例外：撤销(ctrl+z),插入链接(ctrl+K^6)我们可以简单的认为此类操作和word上操作相同^9，皆为ctrl+接下来所需要执行的操作指令

  2. 另外，除了上述和word操作相同的部分，typora还提供了一些word上没有的操作指令，如下面这一些(ctrl+enter)：

    指令 语句
    同一标题(同一表格下换行) ctrl+enter
    建立表格 ctrl+T
    多级标题 ctrl+1~6
    有序列表 ctrl+[
    无序列表 ctrl+]
    降级快捷键^7 Tab
    升级快捷键^7 Tab+Shift
    插入公式 ctrl+Shift+M(math缩写)
    插入代码条 ctrl+Shift+K
    插入图片 ctrl+Shift+I(imagic缩写)
    删除行 ctrl+Shift+删除键
    保存 ctrl+S

上标和下标

  • 上标
    使用^括住内容，如：

    y^2^=4→y^2^=4

  • 下标

    使用~括住内容，如：

    H~2~O→H2O

emjoy表情

  • typora还支持emjoy表情，输入（:+表情的英文单词^8）可以显示出不同的表情。

    如：[:smile:]→:smile:

    ​ [:cry:]→:cry:

    ​ [:email:]→:email:

typora的页内跳转

  • 标题跳转

    格式如下：

    [外部文字](#标题名称)

    举个例子：[点我跳转呀](#一些补充)
    输出的结果为：点我跳转呀（ctrl+点击就可以跳转到对应的地址了）

  • a标签跳转

    对于仅仅使用标题跳转，是无法满足我们使用typora多元化的需求的。所以，我们需要额外引入新的工具：a标签
    typora的官方文档给出了十分详细的介绍，本文只会简单介绍一下a标签的使用，需要深入了解a标签的其他功能可以进入官方文档进行进一步的学习：点此进入

    实际上，a标签的使用并不困难，具体来说的格式是这样子的：

    <a name="标签的变量名">为该标签取的名字</a>.............//跳转之后的地址
<a href="#标签的变量名">为第二个标签取的名字（可与1不同）</a>...........//ctrl+点击后会跳转到上一行的地址

    这样说有点抽象化，我们来举个例子：

    设标签的变量名为bq，两个标签的名字分别叫“一标签”、“二标签”，则命令为：

    <a name="bq">一标签</a>
<a href="#bq">二标签</a>

    最终呈现出来的效果是：

    一标签

    二标签
    ctrl+点击二标签可以跳转到一标签的位置，就酱紫
    a标签也可包裹住地址，如：
    <a name="1">[点此进入](https://cn.overleaf.com/learn/latex/Creating_a_document_in_LaTeX)）
    （注：a标签也可用于访问网站，如<a href="https://www.baidu.com" target="__blank">去百度</a>）
